# Personalized Learning Recommendation System

## Overview
This project is a machine learning-based system that recommends learning content based on user preferences and behavior.

## Features
- Personalized recommendations
- Machine learning model
- Data-driven suggestions

## Tech Stack
- Python
- Pandas
- Scikit-learn

## Project Structure
src/ - source code  
data/ - dataset  
models/ - trained models  

## How to Run
pip install -r requirements.txt  
python app.py
